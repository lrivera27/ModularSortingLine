Prerequisites

What things you need to install the software and how to install them

  1. This projects uses Python 3.6, you can either download it from their website or by downloading the anaconda bundle (recommended).

  2. You need to install OpenCV, since this varies from Operating System to Operating System, we will not be including a guide on how to install it.

  3. You will be needing pyserial for the communication between python and arduino.

     pip install pyserial

  4. For the GUI, the software uses a small library called eel, which can be installed by running:

     pip install eel
